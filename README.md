## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No server or installation required!

### Installation & Running

1. **Clone or download** the repository:
   ```bash
   git clone https://github.com/M7mdQashqesh/book-library.git
   cd book-library
   ```

2. **Open the application**:
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance:
     ```bash
     python -m http.server 8000
     # Then visit: http://localhost:8000
     ```

## ğŸ“– How to Use

1. **Home Page**: View trending and popular books when you first visit
2. **Browse All Books**: Click on "All Books" in the navigation to see the complete catalog
3. **Add to Library**: Click "Add To My Library" on any book to add it to your personal collection
4. **View Personal Library**: Go to "Personal List" to see all your saved books
5. **Remove Books**: Click "Remove From My Library" to delete books from your collection
6. **About Section**: Learn more about our project and mission

## ğŸ’¾ Data Storage

The application uses the browser's **Local Storage** to persist your personal library. This means:
- Your library is saved locally on your device
- Data persists between browser sessions
- Clearing browser cache will reset your library
- No account or server-side storage required
